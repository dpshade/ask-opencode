# Screenshots for Raycast Store

Place screenshots in this directory for Raycast Store submission.

## Requirements

- **Dimensions**: 1440 x 960 pixels
- **Format**: PNG
- **Naming**: `screenshot-1.png`, `screenshot-2.png`, etc.

## Recommended Screenshots

1. **Ask OpenCode** - Main command showing AI response
2. **Agent Selection** - Showing @ agent picker dropdown
3. **Model Selection** - Dropdown with model options
4. **Sessions List** - Recent sessions view
5. **Project Picker** - Project directory selection

## Capture Instructions

1. Open Raycast and use the extension
2. Press `Cmd+Shift+5` for macOS screenshot tool
3. Capture at exactly 1440x960 pixels
4. Save as PNG with descriptive filename
