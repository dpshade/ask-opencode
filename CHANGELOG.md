# Changelog

All notable changes to the OpenCode Raycast extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-26

### Added

- **Ask OpenCode**: Quick coding questions directly from Raycast with streaming responses
- **Agent Selection**: Type `@` to select from available agents (build, plan, etc.)
- **Model Picker**: Dropdown to select AI model from configured providers
- **@path Context**: Use `@~/path/to/project` syntax for project-specific context
- **Path Autocomplete**: Directory suggestions when typing `@path`
- **Session Handoff**: Continue conversations in OpenCode CLI or Desktop app
- **Recent Sessions**: Browse and resume past OpenCode conversations
- **Project Picker**: Quick access to recently used project directories
- **Multi-Terminal Support**: Handoff works with Ghostty, iTerm, Warp, Alacritty, Kitty, Terminal.app, and Hyper
- **Auto-start Server**: Automatically starts OpenCode server if not running
- **Preferences**: Configurable default project, handoff method, terminal app, and auto-start behavior
